BASE DE DONNEE

La base de doonées sont maintenant une partie integrante de notre vie quotidienne que nous ne savons pas souvent que nous utlisons une.


Ex: supermarché, carte de crédit, agence de voyage, biblographique.

OBJECTIFS DU COURS

    Caractériser les systèmesbasés sur les fichiers et les bases de données

    Caractériser les fonctions typiques d'un SGBD

    Connaître les niveaux d'abstraction d'une base de données

    Connaître les niveaux composantes de l'environnement SGBD

    Différencier entre l'administration des données et l'administration de la base de données
